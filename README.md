# Representative Stock Trading Tester
Program to compare stock performance of Representative's transactions vs when the sale is disclosed. Using to find if tracking Representative stock trades is a worthwhile investment strategy: it is.

# Code Explanantion
Calls HouseStockWatcher.com's API to receive an aggregate list of Representative stock transactions. Then uses a for loop to cycle through the stock purchases and calls rep_performance and disclosure_performance functions to find buy prices and current prices of each to calculate performances.

# Conclusions
Representative have a statistically significant overperformance of the market average (~6% yearly). Specific to this program, although Representative overperform vs their diclosure dates, tracking Senator stock purchases from their disclosure dates is a worthwhile investment strategy.
